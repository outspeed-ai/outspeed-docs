## Usage

Use these voice names in your session configuration:

#### JavaScript
```javascript
const sessionConfig = {
  // rest of config...
  voice: "sophie", // Replace with any voice name from the list above
};
```

#### Swift
```swift
let ttsConfig = OutspeedSDK.TTSConfig(voiceId: "sophie")
```
