## Overview

You can use the following voices in your voice agents:

## Stable Voices
- sophie
- savannah
- brooke
- zia
- corinne
- david
- griffin
- carson
- ethan

## Beta Voices
- clark _(beta)_
- linda _(beta)_

<Note>Beta voices are still being improved.</Note>

## Usage

Use these voice names in your session configuration:

```javascript
const sessionConfig = {
  // rest of config...
  voice: "sophie", // Replace with any voice name from the list above
};
```

```swift
let ttsConfig = OutspeedSDK.TTSConfig(voiceId: "sophie")
```
