# Introduction

Outspeed provides low-latency inference and networking infrastructure for speech-to-speech AI models
so developers can build natural, real-time voice experiences.

We offer an **OpenAI Realtime API compatible interface** to interact with models hosted on Outspeed's infrastructure.

## What Outspeed Offers

Outspeed specializes in low-latency, high-quality speech AI technologies:

- **Multi-model Support**: Access to leading speech models including OpenAI, MiniCPM-o, and more
- **Real-time Processing**: Create fluid conversational experiences with minimal latency
- **Developer-friendly Tools**: Comprehensive APIs and testing environments
- **Customization Options**: Configure voices, response parameters, and more
- **Function Calling**: Integrate external tools and services with voice interactions
