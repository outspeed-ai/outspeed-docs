## What you learn?

In this quickstart you'll learn:
1. Add a voice agent to the React frontend of your app
2. Set up a backend endpoint to start a session with the voice agent
3. Show live conversation transcripts on the frontend

## Initial Setup

We'll add the voice agent to your React frontend. You can clone the template [here](https://github.com/outspeed-ai/outspeed-react-template) if you don't have a pre-existing project, or follow the instructions here for an existing project.

### Install Outspeed

Install the React SDK using your preferred package manager:

```bash npm
npm install @outspeed/react
```

```bash pnpm
pnpm add @outspeed/react
```

```bash yarn
yarn add @outspeed/react
```

### Setup session endpoint

<Info>
A session endpoint keeps your API key secure and lets users start voice conversations.
</Info>

Create a `/token` endpoint on your backend. This endpoint should accept a POST request and create a session with the Outspeed API. Here's an example for Express.js:

<CodeGroup>

```javascript Express.js
app.use(express.json());

app.post('/token', async (req, res) => {
  try {
    const { model } = req.body;
    if (model !== 'outspeed-v1') {
      return res.status(400).json({ error: 'Invalid model. Only outspeed-v1 is supported.' });
    }
    const response = await fetch('https://api.outspeed.com/v1/realtime/sessions', {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${process.env.OUTSPEED_API_KEY}`,
        'Content-Type': 'application/json',
      },
      body: JSON.stringify(req.body),
    });
    if (!response.ok) {
      const error = await response.text();
      return res.status(response.status).json({ error: 'Failed to generate token' });
    }
    const data = await response.json();
    res.json(data);
  } catch (error) {
    res.status(500).json({ error: 'Internal server error' });
  }
});
```

```python FastAPI
import os
from fastapi import FastAPI, HTTPException, Request
import httpx

OUTSPEED_API_KEY = os.getenv("OUTSPEED_API_KEY")
app = FastAPI()

@app.post("/token")
async def create_token(request: Request):
    try:
        session_config = await request.json()
        async with httpx.AsyncClient() as client:
            response = await client.post(
                "https://api.outspeed.com/v1/realtime/sessions",
                headers={
                    "Authorization": f"Bearer {OUTSPEED_API_KEY}",
                    "Content-Type": "application/json",
                },
                json=session_config,
            )
            if not response.is_success:
                raise HTTPException(status_code=response.status_code, detail="Failed to generate token")
            return response.json()
    except Exception as e:
        raise HTTPException(status_code=500, detail="Internal server error")
```

</CodeGroup>

### Add a frontend component

Import the required hooks and types from the SDK:

```tsx
import { useConversation } from '@outspeed/react';
import { type SessionConfig } from '@outspeed/client';
```

#### Setup your voice agent


```tsx
const sessionConfig: SessionConfig = {
  model: 'outspeed-v1',
  instructions: 'You are a helpful assistant.',
  voice: 'david', // Voice names must be lowercase. Find more voices at dashboard.outspeed.com
  turn_detection: {
    type: 'semantic_vad',
  },
  first_message: 'Hello! How can I help you today?', // Optional welcome message
};
```

Use the `useConversation` hook to start a session and show live transcripts:

```tsx
const conversation = useConversation({
  sessionConfig,
  getEphemeralKey: async (config) => {
    const res = await fetch('/token', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(config),
    });
    const data = await res.json();
    return data.client_secret.value;
  },
});
```

Display live transcripts in your component:

```tsx
<div>
  {conversation.transcripts.map((t, i) => (
    <div key={i}>{t.text}</div>
  ))}
</div>
```
