<RequestExample>
```jsx
import { useRealtime, RtAudio, RtAudioVisualizer } from "@outspeed/react";
import {useEffect } from "react";

function AppContainer({ config }) {
  const { connection } = useRealtime(config);

  return (
    <div>
        <RtAudio rtConnection={connection} />
        <RtAudioVisualizer
        rtConnection={connection}
        height={100}
        width={100}
        />
    </div>
  );
}
```
</RequestExample>

## Overview

`RtAudioVisualizer` is a React component designed to show audio visualizations into web applications. 


## Props

<ResponseField name="rtConnection" type="object">
  A connection object returned from `useRealtime` hook that can be used to access the underlying HTML audio stream for visualization.
</ResponseField>
