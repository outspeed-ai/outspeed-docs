realtime's Web SDK offers a straightforward API that enables web developers to utilize realtime audio/video capabilities efficiently.

### Installation

To install the package, use:

```bash
npm install @adaptai/realtime-react
```

You can find the source code on [GitHub](https://github.com/xAlpha8/realtime-js)


### Importing

To import the package:

```javascript
import { RealtimeComponent, WebRTCConnectionManager } from "@adaptai/realtime-react";
```

Next, initialize the realtime class by passing your **API Key** to the constructor:

```javascript
const realtime = new RealtimeComponent("your-api-key");
```

To obtain an API key, [Contact us](mailto:contact@getadapt.ai)
