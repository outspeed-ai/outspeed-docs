---
title: "useAvailableMediaDevices"
sidebarTitle: "useAvailableMediaDevices"
description: "A hook that returns all the media devices connected to the system."
---

<RequestExample>
```ts useAvailableMediaDevices.ts
import { useAvailableMediaDevices } from "@outspeed/react"

const {
    availableAudioDevices,
    availableVideoDevices,
} = useAvailableMediaDevices();

```
</RequestExample>

### Returns

<ParamField body="availableAudioDevices" type="MediaDeviceInfo[]">
- Returns a list of all audio input devices connected to the system.
</ParamField>

<ParamField body="availableVideoDevices" type="MediaDeviceInfo[]">
- Returns a list of all video input devices connected to the system.
</ParamField>
