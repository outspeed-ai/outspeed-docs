---
title: "OpenAI Realtime API"
description: "OpenAI's realtime speech-to-speech API"
---

OpenAI's Realtime API provides high-quality speech-to-speech capabilities with low latency. 
This model is available in [Voice Devtools](/get-started/voice-devtools).

It supports:

- Realtime speech conversation
- Tool calling
- Hosted by OpenAI
- High quality audio output
- Supported by [Speech Console](/get-started/speech-console)

For more details, visit [OpenAI's documentation](https://platform.openai.com/docs/guides/realtime).
