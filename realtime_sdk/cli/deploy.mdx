---
title: "realtime deploy"
---

Deploy functions and models to Realtime serverless.

## Usage

```bash
realtime deploy [OPTIONS] FILE_PATH
```

## Options

<Info>To obtain an API key, [Contact us](mailto:contact@getadapt.ai)</Info>

- `--api-key` Your API key. Can be set via the `ADAPT_API_KEY` environment variable.
- `--base-url` Base URL of the Realtime endpoint. Defaults to "https://infra.getadapt.ai".
- `--help` Show help message and exit.

## Arguments

<Warning>
  Currently, we only support single file deployments per Application.
</Warning>

- `FILE_PATH` Path to the Python file containing functions and models to deploy.

## Example

Deploy a function defined in `test.py`:

```bash
realtime deploy --api-key=<your-api-key> test.py
```

## Notes

- If the `--api-key` option is not provided, the CLI will attempt to use the `ADAPT_API_KEY` environment variable.
- The `--base-url` option allows you to specify a custom endpoint for deployment.
- Upon successful deployment, the CLI will provide a function URL that can be used to run the deployed function.
